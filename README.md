# Infix to Postfix & Prefix Converter

##  Overview
This project provides a **Java program** that converts an infix expression into both **postfix** and **prefix** forms.  
It also includes:
- Pseudo code for the algorithm
- Workflow diagram for visual understanding

---

##  Java Program
The main implementation is in `InfixConverter.java`.  
It uses **stack operations** to handle operator precedence and associativity.

---

##  Pseudo Code
The pseudo code is available in `PseudoCode.txt`.  
It outlines the step‑by‑step logic used in the Java program.

---

##  Workflow
The workflow diagram (`WorkflowDiagram.png`) visually explains:
1. Input of infix expression
2. Stack operations for operators and operands
3. Conversion process to postfix and prefix
4. Output of final expressions
